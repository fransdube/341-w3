# 📚 Bookstore API

A comprehensive RESTful API for managing a bookstore collection with full CRUD operations, advanced filtering, pagination, sorting, and search capabilities. Built with Node.js, Express, and MongoDB.

## ✨ Features

- **Complete CRUD Operations** - Create, Read, Update, and Delete books
- **Advanced Filtering** - Filter by genre, price range, and availability
- **Sorting & Pagination** - Sort by any field and paginate results
- **Search Functionality** - Search books by title, author, or genre
- **Data Validation** - Comprehensive input validation with express-validator
- **Error Handling** - Robust error handling for all scenarios
- **MongoDB Integration** - Mongoose ODM with schema validation
- **API Documentation** - Beautiful, interactive HTML documentation
- **Production Ready** - Ready to deploy on Render/Vercel

## 🛠️ Tech Stack

- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - ODM for MongoDB
- **Express-Validator** - Input validation
- **CORS** - Cross-origin resource sharing
- **Dotenv** - Environment variable management

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB Atlas account (free tier works great)
- npm or yarn package manager
- Postman (for testing, optional)
